           Server with Node and Express(Deployable Server)

1.create a server folder
2.inside server folder generate package.json file
   npm init -y
3.Install following packages: express,dotenv,cors,mongoose
   npm i express dotenv cors mongoose
4.create index.js(entry file) file in the server folder
5.We have to create dotenv file to store all the environment variables in the server folder
6.Create .gitignore file in server folder
7.Import dotenv, express cors in index.js
8.create server app using express
9.Use cors and express.json() to your server app
10.create port to listen your server app
11.Run/Listen server app in the specified port
12.Create a routes folder to hold all api request in server folder, create router js file to resolve api request from client api request
   -using express.Router objcet, we define routes for all api
   -Inside each route we have give thelogic to resolve api
13.To define logic to resolveving api, create controller folder, Inside create productController.js file
